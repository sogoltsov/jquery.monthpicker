Linkage with input element

Year/Month:

To associate with an input item, set the monthpicker initialization parameter "input" a reference to that object.

<input class="monthPickerLinkedInput" type="text" value="200101" readonly="readonly"/>
Year/Month: <div id="monthpicker"/>

<script>
    $(document).ready(function() {
        $("#monthpicker").monthpicker({input: $('.monthPickerLinkedInput')});
    });
</script>
